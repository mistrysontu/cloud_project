This project is created for partial fulfilment of the Cloud computing(CS60118) term project by Sontu Mistry (20CS60R15). This project mainly focus how we can deploy multiple ML/DL model/micro services in Cloud(AWS) and verious chalanges associated with it. There are three small micro services running on the same EC2 instance.

  • Todo-list: Keep track of your daily goals.
  • House price prediction: This ML model predict house price based on mentioned parameter.
  • Emotion Detection: This module predict human emotion from an image.

This is a web interface to access all these three micro-services howerver the micro-services can easily be accessed from other interface like android and IOS application.
